# 小票样式

目标不是心理报告，而是一张能复制、能截图、像 `check-please` 一样有热敏纸气质的人格运行小票。

## 硬规则

- 最终小票必须放进 fenced `text` 代码块。
- 默认宽度 48 字符；允许 42、48、56，默认不要超过 48。
- 使用两级分隔线：主分隔线用 `================================================`，副分隔线用 `------------------------------------------------`。
- 必须有票号、日期、来源、明细、总计模式、条码、判词。
- 不要输出普通报告式标题组。字段可以中文，但骨架必须像收据。
- 解释性文字放在小票外；小票内只放票据内容。

## 热敏纸骨架

```text
              <ASCII 类型印章>

              人 格 小 票
================================================
票号        PR-{YYYYMMDD}-{3位序号}
日期        {YYYY-MM-DD}
来源        {记忆 + 缺口问题 / 当前对话 / 用户文本}
范围        {本轮 / 长期记忆 / 低证据娱乐版}
------------------------------------------------
项目                         数值
------------------------------------------------
类型                         {类型原型}
稀有度                       {RARITY}
类型提示                     {TYPE A} / {TYPE B}
主模式                       {一句话模式名}
================================================
运行明细
  能量      {能量来源}
  决策      {决策方式}
  压力      {压力反应}
  启动      {行动入口}
  协作      {协作接口}
------------------------------------------------
五维属性
  自省 {n}  耐心 {n}  混沌 {n}
  洞察 {n}  锋利 {n}
------------------------------------------------
证据
  + {支持证据 1}
  + {支持证据 2}
  - {反证或证据缺口}
------------------------------------------------
使用说明
  适合      {适合的任务}
  避免      {不适合的任务}
  反馈      {需要的反馈}
  启动      {最短启动方式}
================================================
危险提示
  {不要超过两行}
------------------------------------------------
|||| ||| |||| | ||| || ||||
PR-{YYYYMMDD}-{3位序号}

判词
  你不是一种人格。
  你是一套反复出现的运行方式。
```

## 明细区规则

主区像收据，不像档案：

```text
项目                         数值
------------------------------------------------
类型                         企鹅
稀有度                       8.0%
类型提示                     INTJ / INFJ
主模式                       先结构后行动
```

`主模式` 相当于 token 小票里的 `TOTAL`。它是整张人格小票的总计，不要埋在正文里。

## 条码规则

条码用轻量 ASCII，不做二维码。

```text
|||| ||| |||| | ||| || ||||
PR-20260612-001
```

条码下方必须重复票号。票号格式稳定即可，不需要真实全局唯一。

## 语气

- 中文输出。
- 短句。
- 像热敏纸小票，不像长报告。
- 有判断，但不做诊断。
- 可以锋利，但不要羞辱。
- 不使用玄学口吻。
- 不把类型当身份。

## 小票宽度

默认宽度 48 字符。中文宽度在不同终端里不完全一致，优先保证视觉上短、齐、窄。

做法：

- 字段名保持 2-4 个字。
- 值太长就压成一句短话。
- 支持证据最多 3 条。
- 危险提示最多 2 行。
- 判词最多 2 行。

## 低证据版本

证据不足时，仍然输出小票，但把范围改成：

```text
范围        低证据娱乐版
```

并在证据区写：

```text
  - 当前记忆不足，主要依赖本轮回答
```

## 判词备选

根据用户气质选择一句，不要全塞。

```text
你不是一种人格。你是一套反复出现的运行方式。
你不是标签。你是那些没人看见时仍会重复的动作。
类型只是压缩。真正重要的是你怎么启动、怎么卡住、怎么恢复。
```

```text
                 .---.
                (o>o)
               /(   )\
                `---'

              人 格 小 票
================================================
票号        PR-20260612-001
日期        2026-06-12
来源        记忆 + 缺口问题
范围        本轮对话
------------------------------------------------
项目                         数值
------------------------------------------------
类型                         企鹅
稀有度                       8.0%
类型提示                     INTJ / INFJ
主模式                       先结构后行动
================================================
运行明细
  能量      独处整理后恢复
  决策      先看意义和边界
  压力      逃进系统和解释
  启动      先降噪，再开始
  协作      给目标、原因、边界
------------------------------------------------
五维属性
  自省 70  耐心 80  混沌 25
  洞察 65  锋利 40
------------------------------------------------
证据
  + 反复要求先看结构
  + 压力下倾向重建系统
  - 情绪表达材料不足
------------------------------------------------
使用说明
  适合      模糊问题拆解
  避免      无意义忙碌
  反馈      先讲原因，再讲修改
  启动      写下一步，不重写人生
================================================
危险提示
  别把解释清楚误认为已经改变。
------------------------------------------------
|||| ||| |||| | ||| || ||||
PR-20260612-001

判词
  你不是一种人格。
  你是一套反复出现的运行方式。
```
